## Process Mangement
  -  ps: shows the information of progremas that are currently running on computer.
  -  top: shows the compiter resources utilization as well as details about running process.
  -  kill: used to kill the process using PID.
  -  systemctl:  manages the system services and the process, for eg **systemctl status process_name** will show the status of that process wether it is running or stopped.
  -  journalctl: shows the logs of process
  -  pgrep: searches the running process table and prints PIDs of processes whose attributes match a pattern.
  -  pkill: used to kill a process by pattern
  -  killall: used to kill a process by name

## File System
  -  ls: list the files
  -  pwd: display the path of current working directory
  -  mkdir: use to create directory
  -  touch: used to create a file
  -  vim, nano: used to edit a file
  -  cat: show content of the file
  -  chmod: used to update the permission of the file or directly for owner, grpup and other users.

## Networking troubleshooting 
  -  ifconfig: Display all network interface with IP addresses
  -  ifconfig -a: Display all including down netwrok interfaces with IP address
  -  ping host: Send ICMP(Internet Control Message Protocol) echo request to host.
  -  dig domain: display DNS(Domain name system) information
  -  curl LINK: Display the HTML source of LINK
  -  host domain: Display DNS IPS Address for domain
  -  wget LINK: download from the link location
